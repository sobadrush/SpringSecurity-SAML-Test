🔐 Spring Security SAML2 認證測試

歡迎！

這是一個 Spring Security SAML2 認證系統的示範應用程式。

本系統使用 Keycloak 作為 Identity Provider (IDP)。

系統功能：

SAML2 單一登入 (SSO)
Keycloak IDP 整合
用戶屬性自動傳遞
安全登出機制
完整的 SAML Metadata 支援

測試帳號：

  • testuser / password123（一般用戶）
  • admin / admin123（管理員）

💡 IDP: Keycloak (http://localhost:8443)
